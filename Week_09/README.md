本周主要是强化动态规划的练习。在第六章中，对动态规划有个理论认知，动态规划用来解决最优问题，并可淘汰次优解，使用动态规划的关键在于找到状态方程。本章节中进一步认识动态规划，结合本周练习情况，认识到动态规划的难点在于：1）如何将现实问题进行抽象，定义出合适的dp数组，需要多练习、多思考，积累经验；2）若状态拥有更多的维度，比如二维、三维等，需要理清每个维度表示的是什么，在写的时候不要逻辑出错或混乱；3）若状态树很大的时候，需要适当考虑压缩。  

另外，字符串+DP练习时，需要掌握语言的基本常见用法。以Java为例，需要知道String是不可变的;字符串转数组：s.toCharArray()；字符串单个位置值获取：s.charAt(i)；字符串的长度：s.length()；字符串内容对比：s1.equals(s2),s1.equalsIgnoreCase(s2)等等。
